# Controle de Medicamentos

Aplicação CLI em Python para controle de medicamentos e horários.

## Funcionalidades
- Cadastro de medicamentos
- Listagem
- Marcar como tomado
- Integração com API ViaCEP (consulta de endereço por CEP)
- Teste de integração com pytest

## Como executar
```bash
python src/app.py
