# Controle de Medicamentos

## Problema
Muitas pessoas esquecem de tomar medicamentos nos horários corretos.

## Solução
Sistema CLI que permite cadastrar, listar e marcar medicamentos como tomados.

## Público-alvo
Idosos, cuidadores e pessoas com rotina de medicação.

## Funcionalidades
- Adicionar medicamento
- Listar medicamentos
- Marcar como tomado

## Tecnologias
Python, pytest, ruff

## Instalação
pip install -r requirements.txt

## Execução
python src/app.py

## Testes
pytest

## Lint
ruff check .
