# Controle de Medicamentos

Aplicação em Python (CLI) para auxiliar no controle de medicamentos e horários.

## 📌 Funcionalidades
- Cadastro de medicamentos
- Listagem de medicamentos
- Marcar medicamento como tomado
- Busca de endereço por CEP (API ViaCEP)

O projeto utiliza a API pública ViaCEP para buscar informações de endereço a partir do CEP informado.

## 🧪 Testes
O projeto possui testes automatizados utilizando pytest para validar o funcionamento da aplicação.
